# CSV to Portfolio Optimizer

A Python project that cleans historical market data, calculates logarithmic returns, and optimizes portfolios using Markowitz and Michaud models.

## Setup

Download the entire repository from GitHub and move the folder to your Desktop.

Rename the folder exactly:

```text
Optimization tool
```

The folder should contain:

```text
Optimization tool/
├── conv.py
├── optimizer copia 2.py
└── File base.xlsm
```

Install the required Python libraries by opening the Terminal and running:

```bash
python3 -m pip install streamlit pandas numpy openpyxl scipy plotly
```

## Step 1 — Clean the CSV file

First, launch the CSV Market Data Cleaner:

```bash
python3 -m streamlit run "$HOME/Desktop/Optimization tool/conv.py"
```

Upload a CSV file containing dates and historical closing prices.

The cleaner will:

- remove incomplete rows;
- align all assets on common dates;
- calculate logarithmic returns;
- generate a clean Excel file.

Download the Excel file created by the application.

## Step 2 — Run the Portfolio Optimizer

After downloading the clean Excel file, launch the Portfolio Optimizer:

```bash
python3 -m streamlit run "$HOME/Desktop/Optimization tool/optimizer copia 2.py"
```

Upload the clean Excel file generated by `conv.py`.

The optimizer will generate:

- Markowitz efficient portfolios;
- Michaud resampled portfolios;
- efficient frontier charts;
- portfolio allocations;
- expected returns;
- volatility results.

## Workflow

```text
Original CSV
     ↓
CSV Market Data Cleaner
     ↓
Clean Excel File
     ↓
Portfolio Optimizer
     ↓
Portfolio Results
```
